🚀 Nova API Reference v1.0

Complete documentation for Nova Game Engine Lua scripting.

2D Graphics 3D Graphics Audio Keyboard Mouse Gamepad Touch Window System

🎨 2D Graphics

nova_2d_clear(r, g, b)

Clears the screen with a solid color.

nova_2d_clear(10, 10, 40)  -- Dark blue background

nova_2d_rect(x, y, w, h, r, g, b, a?)

Draws a filled rectangle.

ParameterTypeDescription
x, ynumberPosition
w, hnumberWidth and height
r, g, bnumberColor (0-255)
anumber (optional)Alpha (0-255, default 255)
nova_2d_rect(100, 200, 50, 50, 255, 0, 0)       -- Red square
nova_2d_rect(200, 200, 50, 50, 255, 0, 0, 128)   -- Semi-transparent

nova_2d_circle(cx, cy, radius, r, g, b)

Draws a filled circle.

nova_2d_circle(400, 300, 30, 255, 255, 0)  -- Yellow circle

nova_2d_line(x1, y1, x2, y2, r, g, b)

Draws a line between two points.

nova_2d_line(0, 0, 800, 600, 255, 255, 255)  -- Diagonal line

nova_2d_text(text, x, y, size?)

Draws text on screen. size: "small" | "normal" | "big"

nova_2d_text("Hello Nova!", 100, 100)
nova_2d_text("Big Title", 200, 50, "big")
nova_2d_text("Small info", 10, 580, "small")

nova_2d_image(filename, x, y, w?, h?)

Draws a PNG/JPG image from the game folder.

nova_2d_image("player.png", 100, 200)
nova_2d_image("bg.jpg", 0, 0, 1920, 1080)

🧊 3D Graphics

nova_3d_clear(r, g, b)

Clears the 3D viewport.

nova_3d_camera(cx, cy, cz, tx, ty, tz)

Sets camera position and look target.

nova_3d_camera(0, 5, 10, 0, 0, 0)  -- Camera at (0,5,10) looking at (0,0,0)

nova_3d_cube(x, y, z, size, r, g, b)

Draws a colored cube.

nova_3d_cube(0, 0, -5, 1, 255, 100, 50)  -- Orange cube

nova_3d_sphere(x, y, z, radius, r, g, b, slices?, stacks?)

Draws a sphere.

nova_3d_sphere(0, 2, -5, 0.5, 100, 200, 255)
nova_3d_sphere(3, 1, -5, 0.5, 255, 100, 100, 32, 32)  -- High detail

nova_3d_light(x, y, z, r, g, b)

Enables lighting at position.

nova_3d_light(5, 10, 5, 255, 255, 255)  -- White light from above

🔊 Audio

nova_audio_play(filename, volume?)

Plays a sound effect. volume: 0.0 to 1.0

nova_audio_play("jump.wav")
nova_audio_play("explosion.ogg", 0.7)

nova_audio_music(filename, loop?, volume?)

Plays background music.

nova_audio_music("bgm.mp3", true)         -- Loop forever
nova_audio_music("intro.ogg", false, 0.5)  -- Play once, half volume

nova_audio_stop()

Stops music.

nova_audio_pause() / nova_audio_resume()

Pause/Resume music.

nova_audio_volume(v)

Sets master volume (0.0 to 1.0).

⌨️ Keyboard

nova_key_down(key) → bool

Returns true while key is held down.

nova_key_pressed(key) → bool

Returns true only on the frame the key was pressed.

nova_key_any() → bool

Returns true if ANY key is currently down.

KeyDescription
"a" to "z"Letter keys
"0" to "9"Number keys
"f1" to "f12"Function keys
"space", "enter", "escape"Special keys
"up", "down", "left", "right"Arrow keys
"shift", "ctrl", "alt"Modifier keys

🖱️ Mouse

nova_mouse_x() / nova_mouse_y() → number

Absolute mouse position.

nova_mouse_dx() / nova_mouse_dy() → number

Relative mouse movement (for FPS camera).

nova_mouse_wheel() → number

Scroll wheel movement (+1 up, -1 down).

nova_mouse_down(btn) / nova_mouse_pressed(btn) → bool

btn: 0=Left, 1=Middle, 2=Right

nova_mouse_lock(bool)

Locks mouse to window center (FPS mode).

nova_mouse_visible(bool)

Shows or hides cursor.

🎮 Gamepad

Supports up to 4 controllers (Xbox, PlayStation, Generic).

nova_gamepad_count() → number

nova_gamepad_connected(id?) → bool

nova_gamepad_axis(id?, axis) → number (-1.0 to 1.0)

axis: 0=LeftX, 1=LeftY, 2=RightX, 3=RightY, 4=LT, 5=RT

nova_gamepad_down(id?, btn) → bool

btn: 0=A, 1=B, 2=X, 3=Y, 4=Back, 6=Start

nova_gamepad_name(id?) → string

👆 Touch

nova_touch_count() → number

nova_touch_x(id) / nova_touch_y(id) → number

🪟 Window

nova_window_fullscreen(bool)

nova_window_title("text")

nova_window_size() → w, h

nova_window_vsync(bool)

⚙️ System

nova_system_time() → number

Time in seconds since game started.

nova_system_quit()

Immediately exits the game.

nova_print(msg)

Prints to console.

nova_log(msg)

Writes to log file.
